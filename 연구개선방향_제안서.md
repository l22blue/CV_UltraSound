# 🩺 전립선 초음파(TRUS) 인공지능 진단 모델 성능 개선 방안

본 문서는 전립선 초음파(TRUS) 데이터셋 기반의 인공지능 진단 프로젝트를 인계받는 후속 연구자를 위해 작성됨. 최신 의료 인공지능 연구 동향을 바탕으로, 기존 전처리 필터 중심의 선행 연구가 지닌 데이터 구조의 한계를 분석하고 이를 극복하기 위한 3가지 모델 고도화 방안을 제안함.

---

## 1. 최신 의료영상 인공지능 관련 연구 동향
최근 임상적으로 유의미한 전립선암(csPCa) 탐지 연구는 단순 2D 이미지 분류 구조를 탈피하여 다각화되는 추세임.

- **관심 영역 분할 (Segmentation)**: 전립선 기관 영역만을 정확히 분리해 내기 위해 U-Net 및 그 변형 모델 결합이 표준 접근법으로 채택됨.
- **분류 모델 진화 (Classification)**: 전통적인 CNN(ResNet 등) 구조에서 벗어나 3D CNN 혹은 Attention 기법을 도입한 모델 구조로 전환됨.
- **하이브리드 접근법 (DLR)**: 딥러닝 기반 추출 특징과 전통적인 영상 질감 분석(Radiomics) 특징을 병합하는 딥러닝-라디오믹스(DLR) 형태의 하이브리드 앙상블 연구가 지속적인 성과를 보임.

---

## 2. 기존 연구의 한계 및 성능 고도화 제안

기존 활용된 2D CNN (ResNet) 단일 모델 및 단순 영상 전처리 프로세스가 안고 있는 '양성/음성 데이터 불균형 문제'와 '노이즈 식별 한계'를 타개하기 위해 다음 3가지 방법론 중 하나를 후속 연구 방향으로 제안함.

### ① 영상 분할(Segmentation) 모델 기반 관심 영역(ROI) 전처리 파이프라인
- **한계점**: 전체 초음파 화면(배경 및 여백 포함)을 일괄 학습할 경우, 전립선 외 배경 장기 및 글씨 캡션 정보가 노이즈로 작용하여 분류 모델의 위양성(FP)율을 높임.
- **제안 방안**: 전처리 파이프라인의 최전단에 **U-Net 기반 영상 분할 모델**을 배치하여 **전립선(Prostate Gland) 영역 데이터만 추출(Cropping)**함. 이후 해당 관심 영역에 최적화된 노이즈 필터를 적용하여 분류기로 전달함.
- **기대 효과**: 분류 모델로 유입되는 불필요한 배경 형상 및 배경 노이즈가 구조적으로 차단되므로 모델 추론의 목적지향성이 뚜렷해짐. 수치적으로 모델의 민감도(Sensitivity) 개선에 직접적으로 기여함.

### ② Deep Learning Radiomics (DLR) 앙상블 체계 구축
- **한계점**: 한정된 데이터 수량 및 암/정상 분포 비율의 불균형으로 인해, 딥러닝 모델 단독 운용 시 훈련셋에 특화되는 과적합(Overfitting) 발생 가능성이 높음.
- **제안 방안**: 원본 초음파/전처리 영상으로부터 **수학적·통계적 라디오믹스(Radiomics) 특징값(명암 극점, 질감, 형태학적 산포 등)**을 별도 추출함. 이를 딥러닝 신경망이 판별한 로짓(Logit)과 앙상블하여 최종 분류를 수행하는 하이브리드 파이프라인을 구축함.
- **기대 효과**: 블랙박스 성향의 딥러닝 한계를 설명력 있는 수학적 질감 및 통계 특성이 상호 보완함. 데이터 불균형 악조건 속에서도 현저하게 강건한(Robust) 예측 일관성을 확보할 수 있음.

### ③ Attention 기법이 결합된 Vision Transformer (ViT) 실험
- **한계점**: 초기 합성곱 알고리즘 기반 CNN 구조는 이미지 전역의 로컬 픽셀을 동일 계수 비중으로 처리하므로, 초음파 특유의 잔잔한 고주파 반점 노이즈(Speckle) 식별 능력이 취약함.
- **제안 방안**: 분류 알고리즘의 백본(Backbone) 모델 구조를 전통적 CNN에서 글로벌 어텐션 메커니즘 처리에 용이한 **비전 트랜스포머(Vision Transformer, ViT)** 구조로 대체 구성함.
- **기대 효과**: 모델 자체가 노이즈가 산재한 환경 속에서도 암 발현으로 의심되는 픽셀 변형 및 질감 특이점에 높은 가중치를 배분(Attention)하므로, 효율적인 특징 벡터 추출 및 오차율 감소를 기대할 수 있음.

---

## 3. 요약 및 제언
후속 인공지능 성능 고도화의 핵심 열쇠는 파악된 **노이즈 필터링 최적화**와 결부하여, 학습 방해 요소를 근절하는 **U-Net 기반의 영역 정제** 또는 **라디오믹스(Radiomics)**의 결합에 있음. 향후 연구에서는 상기 3가지 접근법 중 실험 환경의 처리 자원과 한계를 고려하여 가장 최적화된 아키텍처를 도입할 것을 권장함.
