# 🩺 전립선 초음파 인공지능 진단 프로젝트

초음파 사진에 다양한 "그림 필터"를 적용해 본 뒤, 인공지능이 암을 찾아내는 능력이 가장 좋아지는 필터를 찾는 프로젝트입니다. 처음 오신 분도 아래 3단계로 쉽게 돌려볼 수 있습니다.

> ⚠️ **잠깐! 꼭 읽어주세요 (데이터 직접 추가 필요)**  
> 깃허브 업로드 용량 및 시간 문제로 **실제 초음파 사진 폴더(`processed/`)와 정답지 엑셀 파일(`TRUS_AI_CRF_forshare.xlsx`)이 현재 제외**되어 있습니다! 코드를 실행하기 전에 반드시 **본인의 파일들을 프로젝트 폴더 안에 직접 넣고** 시작해야 합니다.
## 📂 핵심 파일 요약

> **💡 파이썬(.py) 파일별 역할 안내**

**[공통 도구]**
- `experiment_common.py`: 점수를 채점하고 전체 흐름을 관리하는 뼈대 코드
- `outputs/`: 실험 결과(점수표)가 자동으로 저장되는 폴더

**[전처리 필터 실험 파일들]**
- `baseline.py`: 아무 필터도 씌우지 않은 **[원본]** 사진 실험 (비교 기준점)
- `gaussian.py`: 이미지를 부드럽게 만들어 **자잘한 노이즈를 덮는** 필터
- `median.py`: 영상에 낀 지글지글한 픽셀 반점 노이즈를 **깔끔하게 다듬는** 필터
- `bilateral.py`: 암 판별에 중요한 **경계선은 뚜렷하게 살리면서** 잔잔한 노이즈만 지우는 필터
- `non_local_means.py`: 사진 전체의 패턴을 분석해 **가장 정교하게 노이즈를 닦아내는** 강력한 필터

---

## 🚀 단 3단계! 실험 시작하기

**1단계: 데이터 준비**
- 엑셀 정답지(`TRUS_AI_CRF_forshare.xlsx`)를 폴더 제일 바깥에 둡니다.
- 초음파 사진들은 `processed/` 폴더 안에 모아둡니다.

**2단계: 필수 프로그램 설치**
터미널을 열고 아래 명령어를 입력합니다.
```bash
pip install -r requirements.txt
```

**3단계: 가볍게 1바퀴 실행해보기**
터미널에 아래 명령어를 입력해 학습을 시작합니다.
```bash
# 기본: 원본 사진 테스트
python baseline.py --epochs 1 --batch-size 16

# 예시: 가우시안 필터 테스트
python gaussian.py --epochs 1 --batch-size 16
```
*(다른 필터들도 파일 이름만 바꾸면 바로 실행됩니다!)*

## 📊 결과 확인 (성적표)
학습이 끝나면 `outputs/` 폴더에 생기는 결과 파일(`json`)을 엽니다.
- `accuracy(정확도)` / `auc(전체 성능)` / `sensitivity(암 환자를 실제로 암이라 맞춘 비율)` 값을 보고 어떤 필터가 가장 똑똑한지 비교합니다.
