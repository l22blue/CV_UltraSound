# 🩺 전립선 초음파 인공지능 진단 프로젝트

초음파 사진에 다양한 "그림 필터"를 적용해 본 뒤, 인공지능이 암을 찾아내는 능력이 가장 좋아지는 필터를 찾는 프로젝트입니다. 
경로나 하드웨어(GPU/CPU) 설정이 자동으로 맞춰지도록 내장되어 있어, **이 프로젝트를 처음 접하는 다른 환경의 컴퓨터라도 아래 3단계만 순서대로 따라 하시면 에러 없이 바로 실행할 수 있습니다.**

> ⚠️ **잠깐! 데이터 직접 추가 확인**  
> 깃허브 용량 문제로 **초음파 원본 사진 폴더(`processed/`)와 정답지 엑셀 파일(`TRUS_AI_CRF_forshare.xlsx`)**이 폴더에서 제외되어 있습니다. 아래 1단계를 참고하여 본인의 파일들을 먼저 넣어주세요.

## 📂 핵심 파일 요약

> **💡 각각의 파이썬(.py) 파일은 무엇을 하나요?**

**[공통 도구]**
- `experiment_common.py`: 평가 점수(성적)를 매기고 전체 실험 흐름을 관리하는 뼈대
- `outputs/`: 인공지능이 채점한 최종 성적표(`json`)가 저장되는 결과 폴더

**[전처리 필터 실험 파일들] (실제 우리가 돌려볼 핵심 파일들!)**
- `baseline.py`: **[원본]** 아무 필터도 씌우지 않은 기본 사진으로 실험
- `gaussian.py`: 부드러운 필터로 **자잘한 픽셀 노이즈를 덮고** 실험
- `median.py`: 지글지글한 반점 노이즈들을 **깔끔하게 지우고** 실험
- `bilateral.py`: 암 판별 경계선은 **뚜렷하게 살리되 노이즈만 지우고** 실험
- `non_local_means.py`: 사진 패턴을 분석해 **가장 정교하게 노이즈를 닦고** 실험

---

## 🚀 단 3단계! 흐름대로 실험 시작하기

**1단계: 실험용 데이터 준비 (내가 할 일: 사진과 엑셀 세팅)**
- 수 백장의 초음파 사진들을 `processed/` 라는 이름의 폴더 안에 모아둡니다.
- 암 환자 정답지가 적힌 `TRUS_AI_CRF_forshare.xlsx` 엑셀 파일을 폴더 제일 바깥에 둡니다.
> *(이제 인공지능이 저 사진들과 엑셀 파일을 퍼즐처럼 매칭하며 공부할 준비가 끝났습니다!)*

**2단계: 필수 패키지 설치 (내가 할 일: 분석 도구들 다운로드)**
- 엑셀 파일을 파이썬으로 무사히 읽기 위한 통역기(`openpyxl`) 및 인공지능 학습 기능들을 내 컴퓨터에 설치합니다.
- 터미널을 열고 아래 명령어를 그대로 칩니다.
```bash
pip install -r requirements.txt
```

**3단계: 인공지능 1바퀴 돌려보기 (내가 할 일: 원하는 필터 터미널에서 실행)**
- 컴퓨터 그래픽카드(혹은 노트북 CPU)는 내부에서 알아서 최고 성능으로 자동 인식됩니다! 돌리고 싶은 필터를 골라 바로 실행해 보세요.
```bash
# 기본: 필터 없는 원본사진 학습시키기
python baseline.py --epochs 1 --batch-size 16

# 예시: 가우시안 필터로 사진 다듬고 학습시키기
python gaussian.py --epochs 1 --batch-size 16
```
*(다른 필터들도 파일 이름만 바꾸면 바로바로 똑같이 실행됩니다!)*

## 📊 결과 확인하기 (성적표 분석)
터미널 실행이 무사히 끝나면 `outputs/` 폴더에 생성되는 결과 파일(`json`)을 엽니다.
- 우리는 지금 5가지 파일들로 **어떤 필터가 암 환자를 제일 잘 맞추는지 겨루고 있는 중**입니다! 
- `accuracy(정확도)` / `auc(전체 성능)` / `sensitivity(실제 암 환자를 암이라 알림)` 수치를 중점적으로 서로 비교해 보세요.
