# 🩺 전립선 초음파 인공지능 진단 프로젝트

초음파 사진에 다양한 "그림 필터"를 적용해 본 뒤, 인공지능이 암을 찾아내는 능력이 가장 좋아지는 필터를 찾는 프로젝트입니다. 처음 오신 분도 아래 3단계로 쉽게 돌려볼 수 있습니다.

## 📂 핵심 파일 요약
- `experiment_common.py`: 점수 채점 및 공통 뼈대
- `baseline.py`: [기본] 필터 없는 원본 실험 코드
- `gaussian.py`, `median.py`, `bilateral.py`, `non_local_means.py`: 각각 다른 그림 필터를 적용해보는 테스트 코드들
- `outputs/`: 실험 결과가 자동으로 저장되는 위치

---

## 🚀 단 3단계! 실험 시작하기

**1단계: 데이터 준비**
- 엑셀 정답지(`TRUS_AI_CRF_forshare.xlsx`)를 폴더 제일 바깥에 둡니다.
- 초음파 사진들은 `processed/` 폴더 안에 모아둡니다.

**2단계: 필수 프로그램 설치**
터미널을 열고 아래 명령어를 입력합니다.
```bash
pip install -r requirements.txt
```

**3단계: 가볍게 1바퀴 실행해보기**
터미널에 아래 명령어를 입력해 학습을 시작합니다.
```bash
# 기본: 원본 사진 테스트
python baseline.py --epochs 1 --batch-size 16

# 예시: 가우시안 필터 테스트
python gaussian.py --epochs 1 --batch-size 16
```
*(다른 필터들도 파일 이름만 바꾸면 바로 실행됩니다!)*

---

## 📊 결과 확인 (성적표)
학습이 끝나면 `outputs/` 폴더에 생기는 결과 파일(`json`)을 엽니다.
- `accuracy(정확도)` / `auc(전체 성능)` / `sensitivity(암 환자를 실제로 암이라 맞춘 비율)` 값을 보고 어떤 필터가 가장 똑똑한지 비교합니다.

---

## 🏃‍♂️ 현재 진행 상황 
- ✅ **완료**: 초음파 데이터 세팅 및 AI 학습 뼈대 완성. 원본 포함 **총 5가지 영상 필터 테스트 코드** 작성 완료.
- 🔄 **진행 중**: 5가지 필터들을 각각 테스트하며 **"어떤 방식이 암 환자를 제일 잘 잡아낼까(민감도/AUC 개선)?"** 비교 및 실험 중.
- 🎯 **목표**: 가장 성능이 좋은 필터를 하나 골라 최종 인공지능 모델로 발전시키기.
